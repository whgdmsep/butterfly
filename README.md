# 신문 사설 검색 시스템 (Editorial Search)

이 프로젝트는 신문 사설 데이터를 검색할 수 있는 웹 페이지입니다. 프리미엄 디자인과 다양한 검색 필터(제목, 저자, 주장)를 제공합니다.

## 실행 방법 (How to Run)

브라우저 보안 정책(CORS)으로 인해 로컬 파일을 직접 열면(`file://...`) 데이터를 불러오지 못할 수 있습니다. 아래 방법 중 하나를 사용하여 로컬 서버를 실행해주세요.

### 방법 1: VS Code Live Server 확장 프로그램 사용 (추천)
1. VS Code에서 `index.html` 파일을 엽니다.
2. 우측 하단의 "Go Live" 버튼을 클릭하거나, 파일에서 우클릭 후 "Open with Live Server"를 선택합니다.

### 방법 2: Python 사용
터미널에서 이 폴더로 이동한 후 다음 명령어를 입력하세요:
```bash
# Python 3
python -m http.server 8000
```
그 후 브라우저에서 `http://localhost:8000`으로 접속하세요.

### 방법 3: Node.js (npx) 사용
```bash
npx serve .
```

## 주요 기능
- **다중 검색 필터**: 전체, 제목, 저자, 주장(Argument) 기반 검색
- **반응형 디자인**: PC 및 모바일 최적화
- **실시간 검색**: 입력과 동시에 결과 필터링
- **XML 데이터 연동**: `editor.xml` 파일을 파싱하여 데이터 표시
